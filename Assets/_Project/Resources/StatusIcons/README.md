Put buff, debuff, aura, and death icons here as PNG files.

You can name them naturally. The game tries to understand the filename by keywords.

Examples:

- morte.png, death.png, ko.png, teschio.png
- inibito.png, stun.png, silenzio.png
- dado_giu.png, vigore_down.png, dice_debuff.png
- marcato.png, preda.png, bersaglio.png
- protezione.png, scudo.png
- benedizione.png, bless.png
- furia.png, attacco_buff.png, bonus_attack.png
- spirito.png, ghost.png
- aura_mago.png, aura_mage.png
- aura_paladino.png, aura_scudo.png
- aura_negromante.png, aura_necro.png

If two icons could match the same status, use a clearer name with more keywords, for example:
`aura_mago.png` instead of just `mago.png`.
