# Composable Golem Assets

Generated assets for the miniboss golem material forms.

- `Models/golem_orbit_crystal.obj`: cyan crystal focus with orbiting shards.
- `Models/golem_orbit_glass.obj`: translucent glass panes and halo.
- `Models/golem_orbit_iron.obj`: forged iron chunk, rivets, and hot edges.
- `Models/golem_orbiting_materials_ring.obj`: all three orbiting objects positioned around the golem.
- `Models/golem_hit_crystal_shatter.obj`: crystal damage burst.
- `Models/golem_hit_glass_splinters.obj`: glass damage splinters.
- `Models/golem_hit_iron_sparks.obj`: iron impact sparks and dented plate.

The Unity editor builder `Assets/_Project/Editor/ComposableGolemAssetBuilder.cs` can also create native prefab/material versions from the `Accard N' Die/Rebuild Composable Golem Assets` menu.
